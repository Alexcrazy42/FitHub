Проблема:
Ваш код зависит от внешнего API, которое:
Может быть платным (каждый вызов = деньги).
Нестабильно (падения, таймауты).
Имеет лимиты запросов.
Ещё не разработано (например, API пишет другая команда).


Решение:
Mock-сервер имитирует реальное API, позволяя:

Тестировать код без реальных запросов.
Проверять, как ваш код обрабатывает разные сценарии (200 OK, 404, 500, медленные ответы).
Работать оффлайн (например, в самолёте без интернета).
