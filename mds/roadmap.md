Загрузка файла:
1. добавить запись в бд со статусом Pending
2. попытаться загрузить файл
3. прикрепить в целевую табличку файл, и удалить запись из FileOutbox

если не можем добавить в бд -> ну и ничего не показываем
если не загрузили файл -> BackgroundJob потом удалит этот файл
если не удалось сделать внутри транзакции 3 шаг -> фоновый процесс затрет файл

действия можно сразу делать, без BackgroundJob
