QRTZ_JOB_DETAILS - хранит метаинформацию о джобах: имя, группу, тип .NET, описание, параметры (JobDataMap)
QRTZ_TRIGGERS - общая информация о триггерах: имя, группу, состояние, тип, связанную джобу
QRTZ_SIMPLE_TRIGGERS - Доп. данные для SimpleTrigger (повторы, интервал)
QRTZ_CRON_TRIGGERS - Доп. данные для CronTrigger — выражение CRON, временная зона
QRTZ_SIMPROP_TRIGGERS - Доп. данные для триггеров с расширенными свойствами (расширенные типы)
QRTZ_BLOB_TRIGGERS - Хранит JobDataMap в виде BLOB (если используется сериализация в бинарном виде)
QRTZ_CALENDARS - Хранит календари (например, исключения — праздники)
QRTZ_PAUSED_TRIGGER_GRPS - Группы триггеров, которые приостановлены
QRTZ_FIRED_TRIGGERS - Текущие запущенные триггеры (во время выполнения) — критично для кластера
QRTZ_SCHEDULER_STATE - Состояние каждого экземпляра IScheduler в кластере (имя, время последнего "пульса")
QRTZ_LOCKS - Механизм блокировок (например, TRIGGER_ACCESS,STATE_ACCESS)
QRTZ_PREV_FIRE_TIMES - (Опционально) Предыдущее время запуска — для некоторых триггеров
QRTZ_EXCL_CALENDARS - (Опционально) Исключающие календари
